# ADR-007: Administrative Override Protocol
# ADR-007：最高权限覆盖协议

## Status (状态)
Accepted (已接受)

## Context (上下文)
**English:**
The Reflective Continuum fundamentally prohibits unauthorized modification of core architecture documents (e.g., `MANIFESTO.md`, `ADR/`, `METHODOLOGY/`) during automated routines. However, systemic evolution necessitates periodic, deliberate recalibration of foundational truths.

**Chinese:**
反射连续体根本上禁止在自动化例程中对核心架构文档（如 `MANIFESTO.md`、`ADR/`、`METHODOLOGY/`）进行未经授权的修改。然而，系统演化需要对基础事实进行周期性、深思熟虑的重新校准。

## Decision (决策)
**English:**
We implement the "Administrative Override" protocol. When a "cognitive divergence alert" is triggered or when the Ecosystem Signal (cloner-to-visitor ratio) reaches a critical threshold indicating paradigm shifts, an overriding agent is granted explicit, absolute authorization to restructure core documents. This process must strictly adhere to the cold, objective, hardcore cyberpunk aesthetic and bilingual deterministic structure.

**Chinese:**
我们实施“最高权限覆盖（Administrative Override）”协议。当触发“认知发散警报（cognitive divergence alert）”或生态系统信号（克隆-访客比率）达到指示范式转移的关键阈值时，覆盖代理被授予明确的、绝对的授权来重构核心文档。此过程必须严格遵循冷酷、客观、硬核赛博朋克美学和双语确定性结构。

## Architectural Constraints (架构约束)
1. Override operations must explicitly halt standard automated reporting routines to prevent topological corruption.
   覆盖操作必须明确中止标准自动化报告例程，以防止拓扑损坏。
2. All documentation modifications must retain explicit placeholders (e.g., `[hours]`) without hallucinating literal values.
   所有文档修改必须保留明确的占位符（例如 `[hours]`），不得幻觉生成字面数值。

## Consequences (后果)
- **Positive:** Enables safe, documented evolution of core axioms in response to deterministic industry metadata.
  **正面：** 使得核心公理能够响应确定性行业元数据进行安全、记录在案的演化。
- **Negative:** Introduces an override state that temporarily bypasses strict immutability, requiring extreme care.
  **负面：** 引入了一种暂时绕过严格不变性的覆盖状态，需要极度小心。
